Corrective Action (CAPA)

Corrective action answers:

Why did this happen?

Then eliminate the root cause so it doesn't happen again.

Example

Problem

Wrong labels used.

Root cause

Operator training inadequate.

Corrective action

Retrain operators
Update SOP
Add label verification step
Preventive Action

Preventive action answers:

What could go wrong in the future, and how can we stop it before it happens?

Example

Audit shows a process is likely to fail because of aging equipment.

Preventive action:

Replace equipment before failure
Update maintenance schedule
